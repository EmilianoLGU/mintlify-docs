<Note>
  **Rate Limit**: This endpoint has a default rate limit of **10 products/second** per account. Need higher limits? [Contact our support team](https://cal.com/rivin-ai/30min) - we can accommodate any rate limit requirements on request.
</Note>

<Note>
  **Credit Cost**: This endpoint costs **1 credit** per successful request. Credits are only deducted when the API call succeeds and returns product data. Failed requests (errors, rate limits, etc.) do not consume credits.
</Note>

<RequestExample>

```bash Example UPC Request
curl -X GET "http://rivin.ai/api/walmart/product/details?upc=040094535309" \
  -H "Content-Type: application/json" \
  -H "x-api-key: (YOUR_API_KEY_HERE)"
```

</RequestExample>

### Headers

<ParamField path="x-api-key" type="string" required="false">
  Your unique API key for authentication.

  This key is required in the header of all API requests, to authenticate your account and access Rivin.ai's services. Get your API key through the [**Settings**](https://rivin.ai/settings)\*\* page\*\*.
</ParamField>

<Warning>
  This API only accepts one parameter in the request body of either the: upc, ean, or walmart_product_code. Including multiple parameters or no parameters will result in an error and no data will be retreived.
</Warning>

### Body

<ParamField path="upc" type="string">
  The 12-digit universal product code (upc) that will be used to search Walmart's catalog.
</ParamField>

<ParamField path="ean" type="string">
  The 13-digit european article number (ean) that will be used to search Walmart's catalog.
</ParamField>

<ParamField path="walmart_product_code" type="string">
  The (typically) 6-12 digit walmart product code that will be used to search Walmart's catalog.
</ParamField>

### Response

<ParamField path="upc" type="string">
  The 12-digit universal product code (upc).
</ParamField>

<ParamField path="ean" type="string">
  The 13-digit european article number(ean).
</ParamField>

<ParamField path="walmart_product_code" type="string">
  The (typically) 6-12 digit walmart product code.
</ParamField>

<ParamField path="brand" type="string">
  Enter a description
</ParamField>

<ParamField path="title" type="string">
  Enter a description
</ParamField>

<ParamField path="categories" type="string[]">
  The categories Walmart categorizes the product into. This includes the main category all the way down to sub-categories.
</ParamField>

<ParamField path="product_type" type="string">
  The product type Walmart cateogories the product into. This is usually the last or second to last Walmart category.
</ParamField>

<ParamField path="buybox_price" type="float">
  The current price of the product buybox in USD down to the nearest cent. Returns `null` if no current price is available.
</ParamField>

<ParamField path="customer_rating" type="string">
  The customer rating between 1 to 5 stars. Returns `null` if no customer ratings are available.
</ParamField>

<ParamField path="number_of_reviews" type="string">
  The number of reviews. Returns `null` if no customer reviews are available.
</ParamField>

<Note>
  **Null Values**: Some fields may return `null` when data is not available from Walmart's catalog. This commonly occurs with:

  - Products that are out of stock or discontinued
  - New products without customer reviews yet
  - Products with incomplete catalog information
</Note>
