``` filename
curl --request GET \
  --url http://rivin.ai/api/walmart-product-details \
  --header 'Authorization: Bearer <token>'
```

</RequestExample>

### Headers

<ParamField path="x-api-key" type="string" required={false}>
  Your unique API key for authentication.

  This key is required in the header of all API requests, to authenticate your account and access Rivin.ai's services. Get your API key through the [**Settings**](https://console.anthropic.com/settings/keys)** page**.
</ParamField>

<Warning>
  This API only accepts one parameter in the request body of either the: upc, ean, or walmart_product_code. Including multiple params will result in an error and no data will be retreived.
</Warning>

### Body

<ParamField path="upc" type="string" required={false}>
  The 12-digit universal product code (upc) that will be used to search Walmart's catalog.
</ParamField>

<ParamField path="ean" type="string">
  The 13-digit european article number (ean) that will be used to search Walmart's catalog.
</ParamField>

<ParamField path="walmart_product_code" type="string">
  The (typically) 6-12 digit walmart product code that will be used to search Walmart's catalog.
</ParamField>
