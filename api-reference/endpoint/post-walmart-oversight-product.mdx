<RequestExample>

```bash Example request
curl -X POST "http://rivin.ai/api/walmart/oversight/products" \
  -H "Content-Type: application/json" \
  -H "x-api-key: (YOUR_API_KEY_HERE)" \
  -d '{
    "walmart_product_code": "1370971228"
  }'
```
