## Welcome to Rivin.ai Walmart API

The Rivin.ai Walmart API provides comprehensive access to Walmart's product catalog, including real-time pricing, historical data, customer reviews, and product categorization. Build powerful e-commerce applications with accurate, up-to-date product information.

## **API Overview**

Our API currently offers three main categories of endpoints, with **20+ additional endpoints in active development**:

### **Product Information**
- **GET** `/walmart/product` - Retrieve detailed product information using UPC, EAN, or Walmart product codes
- **GET** `/walmart/product/price-history` - Access historical pricing data for products

### **Oversight Tracking**
- **POST** `/walmart/oversight/product` - Add products to your oversight tracking system
- **DELETE** `/walmart/oversight/product/{id}` - Remove products from oversight tracking

<Note>
  **More Endpoints Coming Soon**: We're actively developing 20+ additional endpoints to expand our Walmart data coverage. Stay tuned for updates!
</Note>

## **Authentication**

All API endpoints require authentication using your API key. Include your key in the `x-api-key` header with every request:

```json
"security": [
  {
    "apiKeyAuth": []
  }
]
```

<RequestExample>

```bash Authentication Example
curl -X GET "https://rivin.ai/api/walmart/product?upc=040094535309" \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY_HERE"
```

</RequestExample>

## **Base URL**

All API requests should be made to:
```
https://rivin.ai/api
```

## **Response Format**

All API responses are returned in JSON format with appropriate HTTP status codes:

- **200** - Success
- **400** - Bad Request (invalid parameters)
- **401** - Unauthorized (invalid or missing API key)
- **404** - Not Found
- **500** - Internal Server Error

## **Rate Limits**

Rate limits are applied per API key to ensure fair usage across all users. Contact our team for custom rate limit configurations for high-volume applications.
